║ 🚀 HOW TO USE IN ADMIN PORTAL                                       ║
╠══════════════════════════════════════════════════════════════════════╣
║                                                                      ║
║ 1. Access Admin Portal                                               ║
║    → http://localhost:8000/admin/login                              ║
║    → Login with admin credentials                                    ║
║                                                                      ║
║ 2. Navigate to WhatsApp Send Page                                    ║
║    → http://localhost:8000/admin/whatsapp-send                      ║
║                                                                      ║
║ 3. Select Users                                                      ║
║    → Check boxes next to users                                       ║
║    → Use search to filter                                            ║
║    → Click "Select All" for bulk                                     ║
║                                                                      ║
║ 4. Compose Message                                                   ║
║    → Type your message                                               ║
║    → Use [[name]] for personalization                                ║
║    → Example: "Hello [[name]], see the attached flyer!"             ║
║                                                                      ║
║ 5. Attach File (Optional) ← NEW!                                     ║
║    → Click "Choose file..." button                                   ║
║    → Select PDF/PNG/JPG/JPEG from your computer                     ║
║    → File preview will appear                                        ║
║    → Click "Remove" if you want to change file                      ║
║                                                                      ║
║ 6. Send                                                              ║
║    → Check confirmation checkbox                                     ║
║    → Click "Send WhatsApp Message"                                   ║
║    → Confirm in popup (shows attachment name)                       ║
║    → Wait for success message                                        ║
║                                                                      ║
