1. Login to Admin Portal
   → http://localhost:8000/admin/login

2. Go to WhatsApp Send Page
   → http://localhost:8000/admin/whatsapp-send

3. Select Users
   ☑ Check boxes next to users
   🔍 Use search to filter
   ✅ Click "Select All" for bulk

4. Compose Message
   📝 Type your message
   👤 Use [[name]] for personalization
   Example: "Hello [[name]], see attached!"

5. Attach File (Optional)
   📎 Click "Choose file..."
   📄 Select PDF/PNG/JPG/JPEG (max 10MB)
   👁️ Preview shows file info
   ✕ Click "Remove" to change

6. Send
   ☑ Check confirmation box
   📤 Click "Send WhatsApp Message"
   ✓ Confirm in popup
   ⏳ Wait for success message
